CASE REAL RESOLVIDO COMPLETO

Problema: Uma empresa de tecnologia precisa calcular o custo de armazenamento de dados. A capacidade de armazenamento dobra a cada nível, mas o custo por unidade diminui pela metade. Calcule o custo total para 5 níveis sabendo que o nível 1 custa R$ 1024,00 por unidade e precisa de 1 unidade.

1 Modelagem do Problema
Capacidade no nível \(n\): \(C_n = 2^{n-1}\) unidades
Custo unitário no nível \(n\): \(P_n = 1024 \cdot 2^{-(n-1)} = 1024 \cdot 2^{1-n}\) reais
2 Cálculo para Cada Nível
Nível 1:
\(C_1 = 2^{1-1} = 2^0 = 1\) unidade
\(P_1 = 1024 \cdot 2^{1-1} = 1024 \cdot 2^0 = 1024 \cdot 1 = \text{R\$ } 1024{,}00\)
Custo: \(1 \times 1024 = \text{R\$ } 1024{,}00\)
Nível 2:
\(C_2 = 2^{2-1} = 2^1 = 2\) unidades
\(P_2 = 1024 \cdot 2^{1-2} = 1024 \cdot 2^{-1} = 1024 \cdot \dfrac{1}{2} = \text{R\$ } 512{,}00\)
Custo: \(2 \times 512 = \text{R\$ } 1024{,}00\)
Nível 3:
\(C_3 = 2^{3-1} = 2^2 = 4\) unidades
\(P_3 = 1024 \cdot 2^{1-3} = 1024 \cdot 2^{-2} = 1024 \cdot \dfrac{1}{4} = \text{R\$ } 256{,}00\)
Custo: \(4 \times 256 = \text{R\$ } 1024{,}00\)
Nível 4:
\(C_4 = 2^{4-1} = 2^3 = 8\) unidades
\(P_4 = 1024 \cdot 2^{1-4} = 1024 \cdot 2^{-3} = 1024 \cdot \dfrac{1}{8} = \text{R\$ } 128{,}00\)
Custo: \(8 \times 128 = \text{R\$ } 1024{,}00\)
Nível 5:
\(C_5 = 2^{5-1} = 2^4 = 16\) unidades
\(P_5 = 1024 \cdot 2^{1-5} = 1024 \cdot 2^{-4} = 1024 \cdot \dfrac{1}{16} = \text{R\$ } 64{,}00\)
Custo: \(16 \times 64 = \text{R\$ } 1024{,}00\)
3 Custo Total
\(\text{Total} = \sum_{n=1}^{5} C_n \cdot P_n = 1024 + 1024 + 1024 + 1024 + 1024\)

\(\text{Total} = 5 \times 1024 = \text{R\$ } 5120{,}00\)
4 Análise e Conclusão

Observamos que cada nível custa exatamente R$ 1024,00, pois:

\(C_n \cdot P_n = 2^{n-1} \cdot 1024 \cdot 2^{1-n} = 1024 \cdot 2^{n-1} \cdot 2^{1-n}\)

\(= 1024 \cdot 2^{(n-1)+(1-n)} = 1024 \cdot 2^0 = 1024 \cdot 1 = 1024\)

Este exemplo demonstra como expoentes negativos e a regra da soma de expoentes se aplicam em situações reais de modelagem de custos.

Conceitos Aplicados:
• Potências positivas e negativas
• Propriedade \(a^m \cdot a^n = a^{m+n}\)
• Expoente zero: \(a^0 = 1\)
• Modelagem matemática de problemas reais
